PONG GAME

Kami membuat game pong sederhana menggunakan ncurses dengan fitur sebagai berikut :
1.	Paddle kiri dikontrol pemain (tombol w naik, s turun, dan tombol q untuk keluar dari game)
2.	Paddle kanan bergerak otomatis 
3.	Bola memantul di atas/bawah dan memantul pada paddle
4.	Jika bola keluar dari kiri/kanan, maka lawan mendapat skor dan bola direset
5.	Skor ditampilkan di pojok kiri/kanan atas layar
6.	Riwayat skor disimpan ke array (10 entri pertama)
7.  Penyimpanan skor ke file RiwayatSkor.txt ketika keluar dari game(tombol q)
